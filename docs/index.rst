Irrigation and Drainage System
========================================

An academic project to implement Irrigation and drainage monitoring System.

It automatic irrigations system based on the weather and soil conditions. It schedules tasks like irrigation based on soil moisture, allowing the farm 
to control everything using their computer/cellphone through a web interface
